How We Use Data

We do not collect or use any user data.

Health Data Access

Safkis reads active calories, weight, height, and birthdate directly from Apple Health (local on-device data via HealthKit framework) to enable calorie tracking. It writes user-logged consumed calories back to Apple Health.

No data is stored, transmitted, or shared with servers or third parties. All processing is on-device.

Sources: Apple HealthKit Documentation.
This app provides no medical advice - consult a healthcare professional.
